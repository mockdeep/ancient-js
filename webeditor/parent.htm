Roark

Making designing fun again

Roark's goal is to make designing fun and simple. No more WYSIWYG editors with 20 different buttons that make you feel stupid.

Simple, easy to use online web editor

Every conceivable service is on the cloud, so why not a decent HTML editor, hence Roark.

Natural language compiler

If you think CSS is a clumsy language to design in (we most certainly agree with you), use Roark, a meta langugae that makes CSS syntax clean and easy to work with.

For example in CSS you'd say : body{background-color:#fff}

In Roark you say : "body bk #fff"

No more brackets or colons. Roark aims to provide the user with a simple, easy to understand, yet powerful meta language for all your designing needs.

How does it work?

Enter in your CSS and HTML in respective tabs and see the results right away.
